## Description
Number guessing game

# Example
Typical games session looks like this:
```terminal
Write your guess
5
User guess: 5
Guess is not correct
Write your guess
6
User guess: 6
Guess is not correct
Write your guess
7
User guess: 7
Guess is correct
Game over: You won
```

## How to run:
```bash
make          # Builds the project
./exec        # Runs the executable
make clean    # Cleans up build files
```
